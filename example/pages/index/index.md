# Redell-UI

> 该`UI`库基于`react hooks` 和 `typescript` 开发; 色彩、样式参考了[林间有风](https://github.com/TaleLin)团队的设计风格;仅供学习和娱乐，请勿在实际生产环境中使用。

## 已完成的组件

- Icon 图标
- Button 按钮
- Input 输入框
- Menu 菜单导航
- message 全局提示
- Modal 对话框
- Sticky 图钉
- cascader 级联选择器
- carousel 轮播图
- grid 栅格
- layout 布局组件
- collapse 折叠面板
- popover 气泡卡片
- upload 文件上传
- slide 图片预览
- loading 加载
