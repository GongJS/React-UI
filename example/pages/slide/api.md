# API

| 属性    | 说明                         | 类型      | 默认值 | 是否必须 |
|---------|------------------------------|-----------|--------|----------|
| visible | 是否渲染图片查看器               | boolean   | false  | 否       |
| list    | 图片链接                     | img[]     | -      | 否       |
| onClose | 用来关闭图片查看器(单击关闭) | () => any | -      | 否       |

## img 
| 属性    | 说明                         | 类型      | 默认值 | 是否必须 |
|---------|------------------------------|-----------|--------|----------|
|  src    | 图片链接 |  string   | -        | 是 |
|  height   | 图片高度 |  string   | -        | 是 |
|  width    | 图片宽度 |  string   | -        | 是 |
