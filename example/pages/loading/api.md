# API

|    属性    |   说明   |    类型    |  默认值  | 是否必须 |
| --------- | ------- | --------- | ------- | -------  |
| name    | loading图标类型 |   string  | 'spin' | 是 |
| size   | 大小 |  'samll' \| 'medium' \| 'large'   | -   | 'small' |
| message   | loading文字 |  string   | - | 否 |
| loading   | 是否显示loading效果 |  boolean   | false | 否 |
