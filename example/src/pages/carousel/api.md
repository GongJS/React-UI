# API

|    属性    |   说明   |    类型    |  默认值  | 是否必须 |
| --------- | ------- | --------- | ------- | -------  |
| type     | 动画效果 |   'fade' \| 'slide'  | 'fade' | 否 |
| autoplay | 是否自动轮播 |  boolean   | true   | 否 |
| interval  | 轮播间隔 |  number   | 3000  单位:毫秒 | 否 |
| height    | 高度 |  string   | 300px | 否 |
