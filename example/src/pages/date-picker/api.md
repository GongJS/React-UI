# API

| 属性         | 说明               | 类型                | 默认值     | 是否必须 |
| ------------ | ------------------ | ------------------- | ---------- | -------- |
| value        | 日期值             | Date                | --         | 否       |
| defaultValue | 日期初始值         | Date                | new Date() | 否       |
| onChange     | 日期发生改变的回调 | (value:Date) => any | ----       | 否       |
| onClose      | 关闭面板的回调     | () => any           | ----       | 否       |
